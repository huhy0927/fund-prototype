私募基金组合功能需求文档

1. 系统概述

1.1 项目名称

私募基金组合管理系统

1.2 项目背景

为满足私募基金组合管理需求，提供组合创建、调仓、业绩分析、持仓管理、归因分析等功能，帮助投资经理高效管理投资组合。

1.3 目标用户

  • 投资经理
  • 基金经理
  • 投资分析师

2. 功能模块

2.1 组合列表

2.1.1 页面入口

导航栏：组合列表

2.1.2 页面布局

  • 顶部操作区：页面标题 + 新建组合按钮
  • 筛选区：组合类型筛选标签 + 搜索框
  • 数据表格：组合列表展示

2.1.3 功能需求

筛选功能：

  • 组合类型筛选：全部、稳健、平衡、进取、CTA
  • 搜索功能：按组合名称搜索

表格字段：

字段名 说明 排序
组合名称组合名称，点击可查看详情支持
组合类型稳健/平衡/进取/CTA支持
创建人组合创建者支持
成立以来收益累计收益率支持
近一周近一周收益率支持
近一月近一月收益率支持
近一年近一年收益率支持
今年以来年初至今收益率支持
操作查看详情、调仓按钮-

分页功能：

  • 显示总条数和当前页码
  • 支持页码切换
  • 每页显示10条

2.2 新建组合

2.2.1 页面入口

  • 导航栏：新建组合
  • 组合列表页：新建组合按钮

2.2.2 页面布局

多步骤向导形式，包含3个步骤：

  1. 基本信息
  2. 选择基金
  3. 配置权重

2.2.3 功能需求

步骤1：基本信息

字段类型必填说明
组合名称文本是组合名称
组合类型下拉选择是稳健/平衡/进取/CTA
初始资金数字是默认1000万，支持自定义
投资目标文本域否组合投资目标描述
备注文本域否其他备注信息

步骤2：选择基金

  • 基金搜索框：支持按基金名称/代码搜索
  • 基金列表展示：基金名称、代码、类型、净值
  • 选择基金：点击"+"按钮添加至已选列表
  • 已选基金列表：显示已选基金，支持删除

步骤3：配置权重

  • 权重输入：每个基金支持输入目标权重
  • 权重校验：单个基金权重0-100%，不强制总和为100%
  • 权重预览：实时显示权重分布

操作按钮：

  • 上一步/下一步：步骤间切换
  • 保存：保存组合配置
  • 取消：放弃创建返回列表

2.4 组合调仓

2.4.1 页面入口

  • 组合列表：调仓按钮
  • 组合详情：调仓按钮

2.4.2 页面布局

顶部信息区：

  • 组合名称
  • 当前净值

调仓表格：

字段说明

基金名称基金名称

3. 数据规范

3.1 组合类型

  • 稳健型
  • 平衡型
  • 进取型
  • CTA型

3.2 基金类型

  • 取基金二级分类数据

3.3 调仓方向

  • 加仓（红色显示）
  • 减仓（绿色显示）

3.4 收益率颜色规范

  • 正收益：红色（#cf1322）
  • 负收益：绿色（#52c41a）

4. 交互规范

4.1 页面导航

  • 顶部导航栏固定
  • 点击导航切换页面
  • 当前页面高亮显示

4.2 表格交互

  • 表头点击排序
  • 行悬停高亮
  • 操作按钮点击响应

4.3 Tab切换

  • 点击Tab切换内容
  • 当前Tab高亮
  • 内容平滑切换

4.4 弹窗交互

  • 点击遮罩关闭
  • 点击X按钮关闭
  • 确认/取消按钮

5. 技术规范

5.1 响应式布局

  • 最小宽度：1200px
  • 最大宽度：1400px
  • 居中显示

5.2 颜色规范

  • 主色：#1890ff（蓝色）
  • 成功：#52c41a（绿色）
  • 危险：#ff4d4f（红色）
  • 背景：#f5f7fa（浅灰）
  • 文字：#333（主文字）、#666（次要文字）

5.3 字体规范

  • 中文字体：Microsoft YaHei
  • 英文字体：-apple-system, BlinkMacSystemFont, Segoe UI
  • 基础字号：14px
  • 标题字号：18px-24px
